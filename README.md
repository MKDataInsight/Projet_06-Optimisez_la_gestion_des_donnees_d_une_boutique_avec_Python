## Scénario

![bottleneck](/assets/bottleneck.png)

Data Analyste chez BottleNeck, un marchand de vin prestigieux, j'ai agrégé les données de stock et les assainir afin de pouvoir les exploiter.

J'ai ensuite présenté mon analyse de ces données au comité de direction.

## Objectifs et compétences
- **Objectif 1** : Effectuer des analyses univariées et multivariées des données pré-traitées
- **Objectif 2** : Pré-traiter des données pour explorer et comprendre leurs caractéristiques
- **Objectif 3** : Traiter, nettoyer des données et définir la gestion des incohérences en conformité du RGPD

**Compétences développées** : 
- Travailler avec un notebook Python
- Effectuer des analyses exploratoires de fichiers
- Effectuer des jonctions de fichiers
- Effectuer des analyses univariées

## Technologies utilisées
- **Python** : Pandas pour manipuler les tables, Matplotlib et seaborn pour les visualisations

## Structure du projet
Le projet est organisé comme suit :
- **Donnees_initiales** : Données des stockes de la boutique
- **Livrables** : Contient le notebook Python
- **presentation.pdf**, **presentation.odp** : PowerPoint présentant ma démarche et mes analyses
